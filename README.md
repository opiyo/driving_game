#DRIVING GO! GO!
![画像1](https://github.com/opiyo/wiki/wiki/img/driving_game.png “DRIVING GO!GO!”)

障害物を避けながら走行距離を伸ばしていくランアクションゲームです。  

#使い方
↑：アクセル  
← →：横移動  
スペース：ジャンプ  

#環境
OS：Mac OS X Yosemite バージョン10.10.2  
Unity：5.2.1f1
